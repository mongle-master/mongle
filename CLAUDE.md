# CLAUDE.md

## 작업 규칙

1. **코드 수정 전 PRD를 확인한다.**

2. **코드 작성 전 mustpass를 꺼내놓는다.** `frontend/docs/mustpass/`, `backend/docs/mustpass/`에 AI가 읽기 좋게 정리한다.

3. **암묵지를 형식지로 다 꺼내놓는다.**

4. **현재 세션에서 구현할 구현 계획은 `frontend/docs/temp/`, `backend/docs/temp/`(gitignore) 폴더에 둔다.**
   왜냐하면 문서에 what이 담기기 시작하면 코드와 문서 간의 drift가 발생하기 때문이다.

5. **암묵지는 주석으로 표시하고, 여러 곳에서 쓰인다면 그건 공통화(리팩토링)할 시그널이다.** 공통화하고 그 코드 파일 내에 주석으로 남긴다.
   왜냐하면 문서는 코드와 가장 가까울수록 좋기 때문이다.

6. **PRD에 나와 있는 내용을 주석에 중복 기술하지 않는다.** 주석은 코드로 설명할 수 없는 암묵지·의도를 기입하는 것이다.
