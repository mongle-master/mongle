# 03. 사람별 타임라인

> **SSOT:** 사람별 타임라인(이벤트 타임블록, 활동 흐름 시각화, 카테고리 필터)의 단일 기준.
> 상위 배경·목표: [개요](./README.md) · 관련 화면: [02 사람 프로필](./02-person.md), [04 기록 작성](./04-record.md)
> 백엔드: [#6 사람별 타임라인 & 활동 흐름 API](https://github.com/mongle-master/mongle/issues/6)

## 1. 목적

특정 사람과 함께한 기록을 시간순으로 보고, 활동 흐름을 회고한다.

## 2. 화면 요소

- 상단 탭: `타임라인` / `프로필` (프로필 탭 → [02](./02-person.md#3-사람-프로필-조회))
- 통계 영역 (함께한 기록 수, 마지막 만남 등 — 계산은 [02 자동 항목](./02-person.md#2-사람-추가--프로필-등록)과 동일 원천)
- **활동 흐름 차트** — 일반 막대/선 차트 대신 관계 기록에 맞는 **다른 형태의 시각화**
- 타임블록 리스트, 각 블록:
  - 이벤트 이름
  - 이벤트 종류
  - 간단한 메모
  - 카테고리 태그 chip
- 하단 내비게이션 (→ [01](./01-home-dashboard.md#9-열린-질문--리스크))
- 인물 추가 버튼 → [02](./02-person.md#2-사람-추가--프로필-등록)
- 상황 기록 작성 버튼 → [04](./04-record.md)

## 3. 상호작용

- 타임블록을 누르면 기록 상세로 이동한다.
- 카테고리 chip을 누르면 해당 유형의 기록만 필터링한다. (카테고리 값 집합은 [04](./04-record.md#4-입력필드-상세-스펙)와 공유)
- `상황 기록 작성`을 누르면 기록 작성 화면으로 이동한다. → [04](./04-record.md)
